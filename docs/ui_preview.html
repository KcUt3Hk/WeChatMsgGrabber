WeChat OCR 简易前端 UI（说明页面，非可交互应用）

该页面仅用于说明与预览，所有操作按钮与标签为静态展示。如需交互操作，请运行桌面应用：ui/simple_gui.py。

重要提示：本页不可交互。要使用“开始/停止扫描、框选、复制命令、保存/加载配置”等功能，请在终端运行桌面 UI。

运行（桌面应用）

在项目根目录运行桌面 UI：

/Users/pankkkk/Projects/Setting/python_envs/bin/python3.12 ui/simple_gui.py

如首次运行提示缺少依赖，请安装 Pillow：

/Users/pankkkk/Projects/Setting/python_envs/bin/python3.12 -m pip install pillow

核心功能

参数设置

窗口标题 聊天区域 x,y,w,h 方向 up/down scroll-delay max-scrolls spm OCR 语言 导出格式 json/csv/md 输出目录 文件前缀 skip-empty verbose

操作

预览聊天区域 框选聊天区域 开始扫描 停止扫描 复制命令 保存配置 加载配置 打开输出目录 退出

权限与提示

  • 请在“系统设置 → 隐私与安全性”中授予屏幕录制与辅助功能权限。
  • 首次加载 OCR 模型耗时较长属正常现象。
  • 如 auto_wechat_scan.py 不支持 txt 格式，请取消勾选 txt。
